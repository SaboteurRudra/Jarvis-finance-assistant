# Jarvis Finance Assistant 📊🤖

Jarvis is a full-stack, AI-powered personal finance application built to help users seamlessly track their expenses and receive intelligent financial insights. 

Built with a **Python/Flask** REST API backend and a responsive, glassmorphic **Vanilla JS/CSS** frontend, Jarvis utilizes local SQLite for secure data persistence. By integrating the **OpenAI API**, the application analyzes spending habits in real-time to act as a personalized financial advisor, providing actionable budgeting advice and summaries.

### Features
- **Dynamic Dashboard**: Track total spending and calculate average expenses dynamically.
- **RESTful Architecture**: Asynchronous Fetch API requests for adding and searching expenses without page reloads.
- **AI Financial Advisor**: Direct integration with OpenAI's LLMs to analyze recent transaction history and provide custom savings tips.
- **Secure Local Storage**: Built on SQLite with parameterized queries ensuring data security and persistence.
- **Premium UI/UX**: Designed with a custom glassmorphism aesthetic, smooth micro-animations, and CSS variables for a modern dark-mode experience.

### Tech Stack
- **Backend**: Python 3, Flask, SQLite3
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **AI Integration**: OpenAI API (`gpt-3.5-turbo`)
